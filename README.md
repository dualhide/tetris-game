# 🎮 テトリス - 楽しいパズルゲーム

小学6年生向けのシンプルで楽しいテトリスゲームです！

## 🎯 遊び方

1. **スタート**ボタンを押してゲーム開始
2. ブロックを操作
   
   **PC・タブレット（キーボード）:**
   - ⬅️ 左矢印：左に動かす
   - ➡️ 右矢印：右に動かす
   - ⬇️ 下矢印：早く落とす
   - ⬆️ 上矢印：回転させる
   - スペース：一番下まで落とす
   
   **スマホ（タッチ）:**
   - 画面下部のボタンで操作
   - 🔄 回転ボタン：ブロックを回転
   - ⬇️ ドロップボタン：一番下まで落とす
   - ⬅️➡️⬇️ 矢印ボタン：移動

3. 横一列にブロックを並べると消えます
4. ブロックが上まで積み上がるとゲームオーバー

## 🌟 特徴

- カラフルで見やすいデザイン
- スコア・レベル・消した行数の表示
- 次に来るブロックが見える
- レベルが上がるとスピードアップ
- スマホでも遊べる
